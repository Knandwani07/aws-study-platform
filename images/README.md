# 🖼 Images

This directory contains visual assets used in the AWS Study Platform project. These images support the documentation, interface design, and presentation of the platform.

## 📂 Purpose

The images stored here are primarily used for:

- Application screenshots
- UI previews
- Feature demonstrations
- Documentation visuals
- Diagrams or architecture illustrations

Keeping images in a dedicated folder helps maintain a clean repository structure and makes it easier to reference visual assets in documentation files such as the main README.
